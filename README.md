# SIMORA

Sistem Informasi Mobil Rental Berbasis Web

## Anggota Tim
- Almelia Restika
- Aurellya Dwinaira Fathania

## Deskripsi
SIMORA adalah sistem informasi rental mobil berbasis web yang membantu pelanggan melakukan pemesanan kendaraan secara online serta membantu admin mengelola data kendaraan, transaksi, dan pelanggan.

## Fitur Utama
- Dashboard
- Booking Mobil
